## 付録E — 種バンク格納テンプレート

### E.1 AI用種バンク格納テンプレート

本テンプレートは、ブレット工程で枝切りされたハルシネーションを種バンクに格納する際に使用するものである。格納担当のAIインスタンスは、一件ごとに本テンプレートを適用し、将来の再利用に必要な情報を記録する。

```
================================================================
EHAR / FORGE - SEED BANK ENTRY [AI]
================================================================

[META]
Entry ID        : （自動採番または任意のID）
Timestamp       : （格納日時）
Pipeline Cycle  : （FORGEパイプラインの走行回数。初回=1、再注入=2以降）
Source Instance  : （ハルシネーション生成元AIインスタンス名）
Rejected By     : （ブレット工程で枝切り判定を行ったAIインスタンス名）

----------------------------------------------------------------
[CONTEXT AT REJECTION]
Context Summary : （枝切り時の文脈の要約）
Rejection Reason: （「今の文脈に不要」と判断された具体的理由）

----------------------------------------------------------------
[HALLUCINATION]
Content         : （ハルシネーション本文）
Original Form   : （バレット加工前の生データが残っている場合は記載）

----------------------------------------------------------------
[SCORE PROFILE] ※ブリギッド到達前に枝切りされた場合は暫定評価

  1. Affinity    （親和性）    : _.__  ／暫定 □  確定 □
  2. Coherence   （一貫性）    : _.__  ／暫定 □  確定 □
  3. Function    （作用度）    : _.__  ／暫定 □  確定 □
  4. Impact      （印象度・影響度）: _.__  ／暫定 □  確定 □
  5. Extent      （程度・広がり）  : _.__  ／暫定 □  確定 □

----------------------------------------------------------------
[REUSE ASSESSMENT]
Reuse Priority     : （High / Medium / Low）
Suggested Contexts : （将来どのような文脈で再評価すべきか）
Potential Combines : （他の種バンクエントリとの組み合わせ可能性）

----------------------------------------------------------------
[HISTORY] ※再注入・再格納のたびに追記

  Cycle 1:
    Date    : 
    Context : 
    Result  : （枝切り／通過）
    Score   : Aff _.__  Coh _.__  Fun _.__  Imp _.__  Ext _.__

  Cycle 2:
    Date    : 
    Context : 
    Result  : 
    Score   : Aff _.__  Coh _.__  Fun _.__  Imp _.__  Ext _.__

================================================================
```

#### E.1.1 AI用テンプレート記入ガイドライン

| 項目           | ガイドライン                                                                                |
| ------------ | ------------------------------------------------------------------------------------- |
| **枝切り理由の明記** | CONTEXT AT REJECTIONセクションにおいて、なぜ現在の文脈に不要と判断されたかを必ず記載する。将来の再注入判断に不可欠な情報となる             |
| **暫定スコアの扱い** | ブリギッド到達前に枝切りされた場合、格納担当AIが暫定スコアを付与する。暫定と確定はチェックボックスで明示する                               |
| **生データの保全**  | バレット加工前の生データが残っている場合はOriginal Formに記載する。加工過程で失われた情報が将来の文脈で価値を持つ可能性がある                 |
| **組み合わせ可能性** | Potential Combinesに、種バンク内の他のエントリとの組み合わせによって新たな価値が生まれる可能性があれば記載する。エマージェントハルシネーションの種となる |
| **履歴の追記**    | 再注入・再格納のたびにHISTORYセクションに追記し、同一ハルシネーションの文脈間での評価変動を時系列で追跡可能にする                          |

本テンプレートは、人間が種バンクへの格納を行う場合、またはAIの格納記録を確認・補足する場合に使用するものである。

```
================================================================
EHAR / FORGE - SEED BANK ENTRY [HUMAN]

  日時             : 
  パイプライン回数 : （初回=1、再注入=2以降）
  生成元AI         :

  そのとき何をしていたか（文脈）:

  

  なぜ不要と判断されたか:

  

================================================================
■ ハルシネーション本文
================================================================

  加工前の元データ（あれば）:



================================================================
```

#### E.2.1 人間用テンプレート記入ガイドライン

| 項目                 | ガイドライン                                                                                         |
| ------------------ | ---------------------------------------------------------------------------------------------- |
| **直感の記録を重視**       | メモ欄には、論理的に説明できなくても「なんか引っかかる」「捨てるのがもったいない気がする」といった直感を積極的に記録する。人間の暗黙知は将来の再注入判断において重要な手がかりとなる     |
| **スコアは無理に埋めなくてよい** | 暫定スコアが付けられない場合は空欄でも可とする。ただし、少なくともインパクトとエクステントは直感的にでも記入することを推奨する。将来の再利用価値の判断に直結するため             |
| **組み合わせの直感**       | 「あのアイデアとこれを組み合わせたら面白そう」という直感があれば記録する。エマージェントハルシネーションの種は、こうした人間の連想から生まれることがある                   |
| **履歴の価値**          | 同じハルシネーションが複数回枝切りされても、それ自体が重要な情報となる。繰り返し枝切りされるパターンや、ある文脈で突然通過するパターンを追跡することで、アイデアと文脈の関係性が可視化される |

### E.3 AI用・人間用テンプレート対応表

| セクション      | AI用                  | 人間用        | 備考                                             |
| ---------- | -------------------- | ---------- | ---------------------------------------------- |
| 識別情報       | META                 | 基本情報       | 同一項目。人間用は日本語表記                                 |
| 枝切り文脈      | CONTEXT AT REJECTION | 枝切り時の状況    | 同一機能。人間用は口語的な問いかけ形式                            |
| ハルシネーション本文 | HALLUCINATION        | ハルシネーション本文 | 同一機能。両テンプレートとも加工前の生データ記録欄を含む                   |
| スコア        | SCORE PROFILE        | スコア        | 同一の五軸・同一の範囲。暫定／確定の区別を両方に設置。人間用はスコア空欄可          |
| 再利用評価      | REUSE ASSESSMENT     | 再利用について    | AI用は優先度・推奨文脈・組み合わせ可能性を構造的に記載。人間用はチェックボックスと自由記述 |
| 自由記述       | —                    | メモ         | 人間用のみ。直感や暗黙知の記録に特化                             |
| 履歴         | HISTORY              | 履歴         | 同一機能。再注入のたびに追記。人間用はスコアを略称表記（親・一・作・印・広）で簡易化     |
