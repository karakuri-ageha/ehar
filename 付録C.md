## 付録C — ハルシネーション生成テンプレートシート

### C.1 AI用ハルシネーション生成テンプレート

本テンプレートは、パイロ工程においてハルシネーション生成担当のAIインスタンスが使用するものである。本テンプレートの目的は、生成されたハルシネーションを記録・追跡可能にすることであり、生成内容そのものを制約することではない。パイロ工程における創造の自由は完全に保障される。

```
================================================================
EHAR / FORGE - PYRO GENERATION SHEET [AI]
================================================================

[META]
Generation ID     : （自動採番または任意のID）
Timestamp         : （生成日時）
Generator Instance: （生成担当AIインスタンス名）
Pipeline Cycle    : （FORGEパイプラインの走行回数。初回=1、再注入=2以降）

----------------------------------------------------------------
[INJECTION CONTEXT]
Injected Context  : （インジェクション工程から受け取った文脈の要約）
Seed Bank Input   : （種バンクからの再注入があった場合、そのEntry ID）

----------------------------------------------------------------
[GENERATION PARAMETERS]
Mode              : （Free / Directed / Collision）
Directive         : （Directedモードの場合、探索方向の指示）
Collision Target  : （Collisionモードの場合、衝突させる対象の情報）

----------------------------------------------------------------
[RAW OUTPUT] ※制約なし。自由に生成する。

  --- Hallucination 1 ---


  --- Hallucination 2 ---


  --- Hallucination 3 ---


  （数量制限なし。生成されただけ追記する）

----------------------------------------------------------------
[GENERATOR NOTES]
Self Observation  : （生成中に気づいたこと・感じた接続・偶発的な発見）
Unexpected Links  : （意図せず生まれた概念間の繋がり）
Heat Level        : （生成の勢い。Cold / Warm / Hot / Inferno）

================================================================
```

#### C.1.1 AI用テンプレート記入ガイドライン

| 項目                     | ガイドライン                                                                                                                    |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| **RAW OUTPUTに制約を設けない** | パイロ工程の本質は創造の自由にある。どれだけ荒唐無稽であっても、どれだけ既存の常識から逸脱していても、すべてを記録する。選別はブレット工程の責務であり、パイロの責務ではない                                    |
| **数量に上限を設けない**         | ハルシネーションの生成数に制限はない。生成されただけRAW OUTPUTに追記する。量と多様性の最大化がパイロ工程の目的である                                                           |
| **生成モードの記録**           | Freeは方向性なしの完全自由生成、Directedは特定の探索方向を指示された生成、Collisionは他のハルシネーションや概念との意図的な衝突による生成を指す。どのモードで生成されたかを記録することで、後続工程や再利用時の参考情報とする |
| **GENERATOR NOTESの活用** | 生成中に偶発的に気づいた接続や発見は、ハルシネーション本体と同等かそれ以上の価値を持つことがある。生成プロセス中のメタ的な観察を積極的に記録する                                                  |
| **Heat Levelの記録**      | 生成の勢いを主観的に記録する。Coldは慎重で控えめな生成、Warmは適度な生成、Hotは活発で大胆な生成、Infernoは制御が難しいほどの爆発的生成を指す。パイロ工程の温度感を可視化する                           |

### C.2 人間用ハルシネーション生成テンプレート

本テンプレートは、人間がパイロ工程に参加する場合に使用するものである。人間もまたハルシネーションの生成者となり得る。論理的な正しさを一切求めない自由な発想を記録するためのフォーマットである。

```
================================================================
EHAR / FORGE - PYRO GENERATION SHEET [HUMAN]
================================================================

■ 基本情報
  生成ID           : 
  日時             : 
  生成者           : 
  パイプライン回数 : （初回=1、再注入=2以降）

■ 受け取った文脈
  （インジェクションから渡された情報を簡潔に記入）



  種バンクからの再注入:  □ あり（Entry ID:        ）  □ なし

■ 生成モード
  □ Free（自由に思いつくまま）
  □ Directed（特定の方向性あり → 方向性:                    ）
  □ Collision（何かとぶつけてみる → 対象:                    ）

================================================================
■ 思いついたこと（制約なし。何でも書く。）
================================================================

  --- 1 ---


  --- 2 ---


  --- 3 ---


  （いくつでも追記可）

================================================================
■ 生成中に気づいたこと
================================================================

  ふと感じた繋がり:



  予想外の発見:



  勢い:  □ おとなしめ  □ まあまあ  □ のってきた  □ 止まらない

================================================================
```

#### C.2.1 人間用テンプレート記入ガイドライン

| 項目              | ガイドライン                                                                       |
| --------------- | ---------------------------------------------------------------------------- |
| **正しさを求めない**    | パイロ工程では論理的な正しさも現実性も一切不要である。「ありえない」「馬鹿げている」と思うことこそ記録する価値がある                   |
| **思いついた順に書く**   | 整理や推敲は不要。思いついた順にそのまま書き出す。バレット工程での加工を前提としているため、断片的でも構わない                      |
| **身体的な感覚も記録する** | 「なんかゾワッとした」「笑えてきた」といった身体的・感情的な反応も、気づいたことの欄に記録する。AIには生成できない情報であり、文脈として高い価値を持つ |
| **勢いの自己評価**     | 生成の勢いを四段階で自己評価する。勢いの記録は、どのような条件下で人間の創造性が活性化するかを追跡する手がかりとなる                   |

### C.3 AI用・人間用テンプレート対応表

| セクション      | AI用                                     | 人間用                              | 備考                                                                     |
| ---------- | --------------------------------------- | -------------------------------- | ---------------------------------------------------------------------- |
| 識別情報       | META                                    | 基本情報                             | 同一項目。人間用は日本語表記                                                         |
| 注入文脈       | INJECTION CONTEXT                       | 受け取った文脈                          | 同一機能。種バンクからの再注入の有無を記録                                                  |
| 生成モード      | GENERATION PARAMETERS                   | 生成モード                            | 同一の三モード（Free / Directed / Collision）。人間用はチェックボックス形式                    |
| ハルシネーション本体 | RAW OUTPUT                              | 思いついたこと                          | 同一機能。両テンプレートとも数量無制限・制約なし                                               |
| メタ観察       | GENERATOR NOTES                         | 生成中に気づいたこと                       | AI用はSelf Observation・Unexpected Links・Heat Levelの三項目。人間用は自由記述と四段階の勢い評価 |
| 生成の勢い      | Heat Level（Cold / Warm / Hot / Inferno） | 勢い（おとなしめ / まあまあ / のってきた / 止まらない） | 同一概念。AI用は英語四段階、人間用は口語的な日本語四段階                                          |
